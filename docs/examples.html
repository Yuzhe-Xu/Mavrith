Examples

Use the repository examples as runnable documentation.

Each example keeps domain behavior in custom blocks, builds a system in Python, validates before running, and prints interpretable outputs.

Run Examples From The Repository

Install development dependencies or point Python at src before running the examples directly. 

uv sync --extra dev --extra yaml
$env:PYTHONPATH = "src"
python -m examples.closed_loop
python -m examples.cruise_control
python examples/export_water_cooling_manifest.py

Common Modifications

Swap the controller

Replace a sampled controller block while preserving ports and SignalSpec contracts.

Add measurement effects

Insert a noise, bias, limiter, or unit-conversion block between plant output and controller input.

Record selected signals

Add an observer that implements on_step(snapshot) and stores only the values needed for review.

After timing changes: update sample_time, offset, and dt together, then run Simulator.validate(...) before simulation.
